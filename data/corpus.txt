Inteligência artificial é a capacidade de máquinas aprenderem e tomarem decisões.
A IA pode ser dividida em IA fraca, que resolve tarefas específicas, e IA forte, que teria capacidades gerais como humanos.
O teste de Turing é um método para avaliar se uma máquina pode exibir comportamento inteligente indistinguível de um humano.
Sistemas especialistas são programas de IA que emulam a capacidade de decisão de um especialista humano.
A IA generativa é capaz de criar novos conteúdos como texto, imagens, música e código.
Redes neurais artificiais são inspiradas no funcionamento do cérebro humano.
O aprendizado por reforço treina agentes através de recompensas e penalidades.
Machine learning é um subcampo da inteligência artificial focado em aprendizado por dados.
Existem três tipos principais de aprendizado de máquina: supervisionado, não supervisionado e por reforço.
No aprendizado supervisionado, o modelo é treinado com dados rotulados.
O aprendizado não supervisionado encontra padrões em dados sem rótulos predefinidos.
Algoritmos de classificação preveem categorias, enquanto regressão prevê valores numéricos.
Overfitting ocorre quando o modelo se ajusta demais aos dados de treino e generaliza mal.
Cross-validation é uma técnica para avaliar a capacidade de generalização do modelo.
Feature engineering é o processo de criar variáveis relevantes para melhorar o modelo.
Gradient descent é um algoritmo de otimização usado para minimizar a função de erro.
Deep learning utiliza redes neurais profundas para processamento de informações complexas.
Redes neurais convolucionais (CNN) são especializadas em processamento de imagens.
Redes neurais recorrentes (RNN) processam sequências de dados como texto e séries temporais.
LSTM (Long Short-Term Memory) resolve o problema de dependências de longo prazo em RNNs.
Transformers revolucionaram o NLP com mecanismo de atenção e processamento paralelo.
GPT (Generative Pre-trained Transformer) é uma arquitetura para geração de texto.
BERT (Bidirectional Encoder Representations from Transformers) é usado para compreensão de linguagem.
Autoencoders são redes que aprendem representações comprimidas dos dados.
GANs (Generative Adversarial Networks) geram dados realistas através de duas redes competindo.
Transfer learning permite usar modelos pré-treinados para novas tarefas.
Processamento de linguagem natural permite que computadores entendam linguagem humana.
Tokenização é o processo de dividir texto em unidades menores como palavras ou subpalavras.
Stemming reduz palavras à sua raiz, enquanto lematização encontra a forma canônica.
Stopwords são palavras comuns como artigos e preposições que geralmente são removidas.
Bag of Words representa texto como frequência de palavras, ignorando ordem e gramática.
TF-IDF (Term Frequency-Inverse Document Frequency) pondera a importância das palavras em documentos.
Word embeddings como Word2Vec e GloVe representam palavras como vetores densos.
Named Entity Recognition (NER) identifica entidades como pessoas, lugares e organizações.
Part-of-Speech tagging classifica palavras em categorias gramaticais.
Dependency parsing analisa a estrutura gramatical e relações entre palavras.
A similaridade de cosseno mede o ângulo entre vetores para comparar documentos.
A análise de sentimento identifica emoções em textos como positivo, negativo ou neutro.
VADER é um analisador de sentimento otimizado para textos de redes sociais.
A polaridade de sentimento varia de -1 (muito negativo) a +1 (muito positivo).
Análise de sentimento baseada em aspectos identifica sentimentos sobre características específicas.
Detecção de ironia e sarcasmo é um desafio para sistemas de análise de sentimento.
Sentiment lexicons são dicionários com palavras e suas polaridades de sentimento.
A análise de sentimento é usada para monitorar marca, opinião pública e satisfação do cliente.
Chatbots são programas que simulam conversas humanas usando técnicas de NLP.
Chatbots baseados em regras seguem scripts predefinidos para responder.
Chatbots baseados em recuperação selecionam respostas de uma base de conhecimento.
Chatbots generativos criam respostas originais usando modelos de linguagem.
Intenção é o objetivo do usuário, enquanto entidades são informações relevantes na mensagem.
Slot filling é o processo de extrair informações necessárias para completar uma tarefa.
Contexto conversacional mantém informações ao longo de múltiplas interações.
Fallback é a resposta padrão quando o chatbot não entende a mensagem.
Reconhecimento de voz converte áudio falado em texto escrito.
Speech-to-Text (STT) transcreve fala em texto, enquanto Text-to-Speech (TTS) sintetiza voz.
Modelos acústicos mapeiam sinais de áudio para unidades fonéticas.
Modelos de linguagem estimam a probabilidade de sequências de palavras.
Whisper da OpenAI é um modelo robusto de reconhecimento de voz multilíngue.
Voice Activity Detection (VAD) identifica quando há fala no áudio.
Speaker diarization separa diferentes falantes em uma gravação de áudio.
Ruído de fundo e sotaques são desafios para sistemas de reconhecimento de voz.
Tradução automática usa modelos de linguagem para converter texto entre idiomas.
Tradução neural (NMT) usa redes neurais profundas para tradução de alta qualidade.
Modelos multilíngues como mBERT e XLM-RoBERTa processam múltiplos idiomas.
Zero-shot translation traduz entre pares de idiomas não vistos durante o treinamento.
Code-switching é a alternância entre idiomas em uma mesma conversa.
Detecção de idioma identifica automaticamente a língua do texto de entrada.
Transliteração converte texto entre diferentes sistemas de escrita.
O Python é uma linguagem popular para desenvolvimento de aplicações de IA.
TensorFlow e PyTorch são frameworks amplamente usados para deep learning.
Scikit-learn oferece algoritmos de machine learning tradicionais em Python.
NLTK (Natural Language Toolkit) é uma biblioteca fundamental para NLP em Python.
SpaCy é uma biblioteca industrial para processamento de linguagem natural.
Hugging Face Transformers fornece modelos pré-treinados de última geração.
Pandas e NumPy são essenciais para manipulação de dados em Python.
Jupyter Notebooks permitem desenvolvimento interativo e experimentação.
MLflow e Weights & Biases ajudam a rastrear experimentos de machine learning.
FastAPI e Flask são usados para criar APIs de modelos de IA.
AWS Lambda executa código sem gerenciar servidores (serverless).
Amazon SageMaker é uma plataforma para construir, treinar e implantar modelos de ML.
Amazon Comprehend oferece NLP como serviço, incluindo análise de sentimento.
Amazon Transcribe converte fala em texto automaticamente.
Amazon Polly sintetiza fala natural a partir de texto.
Amazon Lex permite criar interfaces conversacionais com voz e texto.
Containers Docker empacotam aplicações com todas as dependências.
ECR (Elastic Container Registry) armazena imagens de containers na AWS.
API Gateway gerencia APIs RESTful e WebSocket na AWS.
CloudWatch monitora logs e métricas de aplicações na AWS.
Viés algorítmico ocorre quando modelos de IA perpetuam preconceitos dos dados de treino.
Explicabilidade em IA busca tornar decisões de modelos compreensíveis para humanos.
Privacidade de dados é uma preocupação central no desenvolvimento de sistemas de IA.
IA responsável considera impactos sociais, éticos e ambientais da tecnologia.
Regulamentações como LGPD e GDPR afetam como dados são usados em sistemas de IA.
Alucinação em IA generativa refere-se à geração de informações falsas ou inventadas.
Human-in-the-loop mantém supervisão humana em decisões críticas de IA.
